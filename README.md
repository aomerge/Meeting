# Meeting
thi is the API for meeting where you can create a meeting and invite people to join the meeting.

## Requirements
- docker
- docker-compose
- Java 17
- Maven
- Node 20

## Rute to run the project
- Clone the project.
- Run the command `docker-compose up` to start the database.
